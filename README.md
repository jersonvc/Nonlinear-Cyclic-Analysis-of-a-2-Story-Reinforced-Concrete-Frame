# Nonlinear Cyclic Analysis of a 2-Story Reinforced Concrete Frame

This repository contains a high-fidelity, advanced numerical simulation pipeline implemented in **OpenSeesPy** to evaluate the nonlinear seismic behavior of a 2-story, 1-bay reinforced concrete (RC) frame. The model employs a distributed plasticity formulation with fiber-discretized cross-sections to track the spread of plasticity, core concrete confinement effects, and steel hysteretic degradation under large cyclic displacement demands.

---

## Technical Specifications & Framework Topology

The structural model is developed in a two-dimensional Cartesian coordinate system with 3 degrees of freedom (DOFs) per node. The frame layout represents a typical moment-resisting frame designed under strong-column/weak-beam guidelines, adapted to a long-span scenario.

### Geometrical Layout

* **Total Frame Width (Bay Span, $L$):** 6000 mm
* **Story Height ($H_{\text{story}}$):** 3000 mm (Total Height = 6000 mm)
* **Column Cross-Section:** 400x400 mm square section (8$\phi$20 longitudinal reinforcement)
* **Beam Cross-Section:** 400x600 mm deep rectangular section (8$\phi$20 longitudinal reinforcement)
* **Clear Concrete Cover:** 40 mm to the longitudinal bar centroid

### Material Constitutive Models

The framework maps mechanical behavior using uniaxial material laws grounded in experimental continuum mechanics:

| Structural Component | OpenSees Material Type | Key Mechanical Parameters |
| --- | --- | --- |
| **Confined Core Concrete** | `Concrete02` (Kent-Park modified) | $f'_c = -48\text{ MPa}$, $\varepsilon_0 = -0.004$, $f'_{cu} = -10\text{ MPa}$, $\varepsilon_{cu} = -0.0125$ |
| **Unconfined Cover Concrete** | `Concrete02` (Linear tension softening) | $f'_c = -40\text{ MPa}$, $\varepsilon_0 = -0.002$, $f'_{cu} = 0\text{ MPa}$, $\varepsilon_{cu} = -0.005$ |
| **Longitudinal Rebar Steel** | `Steel02` (Giuffré-Menegotto-Pinto) | $f_y = 400\text{ MPa}$, $E_s = 200,000\text{ MPa}$, $b = 0.015$ (Strain Hardening) |

## First-Principles Analytical Benchmarks

To validate the numerical outputs generated by the OpenSeesPy non-linear solver, an upper-bound plastic mechanism analysis is performed based on kinematic plastic collapse theory.

### 1. Cross-Sectional Plastic Moment Capacities

The ultimate moment capacities are determined assuming full plastic plastification of the reinforcement steel layers:

* **Beam Plastic Moment ($M_{p,\text{beam}}$):**
Assuming zero net axial force in the beam during pure lateral displacement sway:

$$M_{p,\text{beam}} = A_{st} \cdot f_y \cdot d'$$



Where tension steel area $A_{st} = 3 \times 314.16\text{ mm}^2 = 942.5\text{ mm}^2$, and the internal rebar distance $d' = 520\text{ mm}$:

$$M_{p,\text{beam}} = 942.5\text{ mm}^2 \times 400\text{ MPa} \times 520\text{ mm} \times 10^{-6} \approx 196.0\text{ kN}\cdot\text{m}$$


* **Column Plastic Moment ($M_{p,\text{col}}$):**
Accounting for the second-order contribution of the constant gravity axial force ($P_{\text{axial}} = 0.10 \times 40\text{ MPa} \times 160,000\text{ mm}^2 = 640\text{ kN}$):

$$M_{p,\text{col}} = \frac{A_{st} \cdot f_y \cdot d' + |P_{\text{axial}}| \cdot \left(\frac{B_{\text{col}}}{2} - \frac{\text{cover}}{2}\right)}{10^6} \approx 235.8\text{ kN}\cdot\text{m}$$



### 2. Governing Plastic Collapse Mechanism

Equating external work done by the base shear lateral force $V_n$ to the internal virtual work dissipated through the formation of plastic hinges gives the theoretical system capacity:

$$V_n = \frac{M_{p,\text{col}} + 2 \cdot M_{p,\text{beam}}}{H_{\text{story}}} = \frac{235.8 + 2(196.0)}{3.0} = 209.3\text{ kN}$$

## Repository Structure & Numerical Implementation

### Fiber Section Strategy

The sections are discretized using a fiber approach where each rectangular sub-region (`patch quad`) tracks individual stress-strain states during cyclic loading.

* The column utilizes a tightly confined rectangular core matrix surrounded by four independent unconfined concrete cover patches.
* The beam section rotates its depth into the local coordinate system plane to correctly evaluate flexural performance under dominant horizontal excitation.

### Simulation Routine

1. **Gravity Phase:** Constant axial loads corresponding to a $10\%$ column axial load ratio are incrementally applied at the top story nodes via an explicit load-controlled static analysis routine.
2. **Cyclic Displacement Phase:** The gravity state is locked, and a multi-step cyclic displacement protocol is executed at the roof level (Node 5) under quasi-static displacement control. The protocol steps through increasing drift ratios: **0.5%, 1.0%, 1.5%, 2.0%, and 3.0%**, with 3 complete cycles per amplitude level.

> ### Limitations of the Current Model
> 
> 
> 1. **Shear-Flexure Interaction:** Elements are governed by pure non-linear flexural formulations; explicit shear failure modes and concrete diagonal tension degradations are omitted.
> 2. **Fixed-End Rotations:** Bond-slip deformations within the beam-column joints are modeled as rigid connections. For highly degradation-dominated regional checks, implement zero-length rotational spring elements using `uniaxialMaterial BarSlip`.
